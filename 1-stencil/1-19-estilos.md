# 1-19 Estilos

```jsx
@Component({
  tag: 'auna-text',
  styleUrl: 'auna-text.css'
})
```

```scss
// css aplicado al HOST
auna-text {
  width: 100px;
}
```

```jsx
@Component({
  tag: 'auna-text',
  styleUrl: 'auna-text.css',
  shadow: true
})
```

```scss
// :HOST
:host {
  width: 100px;
}
```
