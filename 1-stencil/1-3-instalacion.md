# 1-3 Instalacion

## Para una instalacion limpia

```text
npm init stencil
```

## O Actualizacion a un proyecto creado

Se debe instalar la dependencia de Stenciljs en el package.json

```text
npm install @stencil/core@latest --save-exact
```
